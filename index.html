Music Prediction using unsupervised learning Technology

This project leverages Spotify data for generating song recommendations. Utilizing K Means Clustering, songs are grouped into clusters based on their characteristics. PCA is incorporated to reduce dimensionality. The clustered and reduced data is then integrated into a website for providing personalized song recommendations. The website includes a feedback system where user input contributes to refining and enhancing the recommendation results over time.

Invasive Ductal Carcinoma Detection using Deep learning

I contributed to a project using Deep Ensemble Neural Networks and transfer learning to detect invasive ductal carcinoma in breast histopathology images. With a dataset of 160,000 50x50 pixel images, we utilized Tensor Processing Units (TPUs) for a 10x computation speedup. Fine-tuning the data with multiple models achieved a 96% accuracy rate in predicting invasive ductal carcinoma. The project prioritized computational efficiency with TPUs, showcasing a comprehensive approach to model development and infrastructure optimization in breast cancer detection.
